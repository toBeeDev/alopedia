# 📖 ScalpCheck 용어 정의 (Glossary)

> 모든 에이전트가 공유하는 프로젝트 공통 용어집. 커뮤니케이션 혼선을 방지합니다.

---

## 1. 의료/도메인 용어

| 용어                                              | 정의                                                      | 비고                           |
| ------------------------------------------------- | --------------------------------------------------------- | ------------------------------ |
| **노우드-해밀턴 스케일** (Norwood-Hamilton Scale) | 남성형 탈모의 진행 단계를 7단계로 분류하는 국제 표준 척도 | 본 서비스는 5등급으로 단순화   |
| **Type I~VII**                                    | 노우드 스케일의 원래 7단계 분류                           | 1=정상, 7=광범위 탈모          |
| **M자 탈모**                                      | 이마 양 옆 헤어라인이 후퇴하여 M자 형태가 되는 패턴       | Type III~IV에 해당             |
| **정수리 탈모**                                   | 두정부(정수리) 영역의 모발이 빠지는 패턴                  | Type IV+                       |
| **헤어라인** (Hairline)                           | 이마와 모발의 경계선                                      | 전면이마 촬영의 핵심 분석 영역 |
| **모발 밀도** (Hair Density)                      | 단위 면적당 모발 수                                       | AI 분석 항목                   |
| **모발 굵기** (Hair Thickness)                    | 개별 모발의 직경                                          | AI 분석 항목 (연모화 여부)     |
| **연모화** (Miniaturization)                      | 모발이 점차 가늘어지는 현상                               | 탈모 진행의 핵심 지표          |
| **두피 상태** (Scalp Condition)                   | 두피의 색, 각질, 염증 여부                                | AI 분석 항목                   |
| **피나스테리드** (Finasteride)                    | DHT 억제 탈모 치료 약물 (경구)                            | 약물 후기 게시판 주요 태그     |
| **미녹시딜** (Minoxidil)                          | 혈관 확장 탈모 치료 약물 (외용)                           | 약물 후기 게시판 주요 태그     |
| **두타스테리드** (Dutasteride)                    | DHT 억제 탈모 치료 약물 (경구)                            | 피나스테리드 대안              |
| **모발이식** (Hair Transplant)                    | 후두부 모낭을 탈모 부위에 이식하는 시술                   | 시술 후기 게시판               |
| **메조테라피** (Mesotherapy)                      | 두피에 영양 성분을 직접 주입하는 시술                     | 시술 후기 게시판               |
| **PRP** (Platelet-Rich Plasma)                    | 자가 혈소판 풍부 혈장 주입 시술                           | 시술 후기 게시판               |

## 2. 서비스 용어

| 용어                          | 정의                              | 코드 참조             |
| ----------------------------- | --------------------------------- | --------------------- |
| **스캔** (Scan)               | 한 번의 촬영 세션 (3장 세트)      | `scans` 테이블        |
| **분석** (Analysis)           | AI가 스캔을 분석한 결과           | `analyses` 테이블     |
| **등급** (Grade)              | AI 분석 결과의 1~5 단계 분류      | `norwood_grade` 컬럼  |
| **점수** (Score)              | AI 분석의 세부 수치 (0.0~100.0)   | `score` 컬럼          |
| **타임라인** (Timeline)       | 시간순 분석 기록 뷰               | Dashboard 기능        |
| **비교뷰** (Compare View)     | 두 시점 사진을 나란히 비교        | Before/After 슬라이더 |
| **스트릭** (Streak)           | 연속 기록 일수                    | `streak_current`      |
| **치료 로그** (Treatment Log) | 약물/시술 복용·시행 기록          | `treatments` 테이블   |
| **카피** (Copy)               | 사용자에게 보이는 모든 텍스트     | UX Writer 관할        |
| **면책 고지** (Disclaimer)    | 의료 진단 아님을 알리는 법적 고지 | 모든 분석 화면        |

## 3. 기술 용어

| 용어                                      | 정의                                             | 사용 맥락               |
| ----------------------------------------- | ------------------------------------------------ | ----------------------- |
| **EXIF** (Exchangeable Image File Format) | 사진에 포함된 메타데이터 (GPS, 디바이스 정보 등) | 업로드 시 반드시 제거   |
| **RLS** (Row Level Security)              | PostgreSQL 행 단위 접근 제어 정책                | Supabase 보안 핵심      |
| **PWA** (Progressive Web App)             | 앱처럼 동작하는 웹 앱                            | 모바일 설치 가능        |
| **SSR** (Server-Side Rendering)           | 서버에서 HTML을 렌더링하여 전달                  | Next.js App Router      |
| **Route Handler**                         | Next.js의 서버사이드 API 엔드포인트              | Gemini API 프록시       |
| **Edge Function**                         | CDN 엣지에서 실행되는 서버리스 함수              | Supabase Edge Functions |
| **Magic Bytes**                           | 파일 시작 부분의 식별자 바이트                   | 이미지 유효성 검증      |
| **Signed URL**                            | 제한된 시간 동안만 유효한 인증된 URL             | 이미지 접근 제어        |

## 4. 등급 정의 (서비스 내 표준)

| Grade | 서비스 라벨       | 노우드 매핑 | 컬러      | 권장 액션             |
| ----- | ----------------- | ----------- | --------- | --------------------- |
| 1     | 🟢 정상           | Type I      | `#22C55E` | 기록 지속             |
| 2     | 🟡 경미           | Type II     | `#EAB308` | 정기 관찰             |
| 3     | 🟠 주의           | Type III    | `#F97316` | 관리 시작 권유        |
| 4     | 🔴 관리 필요      | Type IV~V   | `#EF4444` | 전문의 상담 권유      |
| 5     | 🟣 전문 상담 추천 | Type VI~VII | `#A855F7` | 전문의 상담 강력 권유 |

> ⚠️ 등급 4~5에서는 병원 방문 권유 팝업이 반드시 표시되어야 합니다.
