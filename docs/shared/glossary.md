# 용어 정의 (Glossary)

> 프로젝트 전체에서 통일된 용어를 사용하기 위한 정의서

---

## 도메인 용어

| 용어              | 영문              | 정의                                                         |
| ----------------- | ----------------- | ------------------------------------------------------------ |
| 노우드-해밀턴     | Norwood-Hamilton  | 남성형 탈모 진행 단계를 7단계로 분류하는 국제 표준 척도      |
| 스캔              | Scan              | 두피 사진 3장 세트(정수리·전면·측면)를 촬영하는 1회 세션     |
| 분석              | Analysis          | Gemini Vision API를 통해 스캔 이미지를 처리한 AI 결과물      |
| 등급              | Grade             | 노우드-해밀턴 기반 간소화 5등급 (1=정상 ~ 5=전문 상담 추천)  |
| 점수              | Score             | 0.0~100.0 범위의 두피 상태 수치 (높을수록 건강)              |
| 스트릭            | Streak            | 연속으로 스캔을 기록한 일수                                  |
| 치료 로그         | Treatment Log     | 사용자가 기록하는 약물·시술·보조제 이력                      |
| 타임라인           | Timeline          | 날짜순 스캔 기록 + 점수 변화 그래프                          |

## 게시판 용어

| 용어              | 코드               | 정의                                           |
| ----------------- | ------------------ | ---------------------------------------------- |
| 약물 후기         | medication_review  | 피나스테리드, 미녹시딜, 두타스테리드 등 후기    |
| 시술 후기         | procedure_review   | 모발이식, 메조테라피, PRP 등 후기              |
| Q&A               | qna                | 탈모 관련 질문 게시판 (채택 시스템)            |
| 라운지            | lounge             | 자유게시판                                     |
| 채택              | Adoption           | Q&A에서 질문자가 최적 답변을 선택하는 기능     |

## 기술 용어

| 용어              | 정의                                                         |
| ----------------- | ------------------------------------------------------------ |
| EXIF              | 이미지 파일에 포함된 메타데이터 (GPS, 디바이스 정보 등)      |
| RLS               | Row Level Security, Supabase/PostgreSQL의 행 단위 접근 제어  |
| Magic Bytes       | 파일 시작 부분의 바이트 패턴으로 실제 파일 형식을 판별       |
| Route Handler     | Next.js App Router의 서버사이드 API 엔드포인트               |

## UI 용어

| 용어              | 정의                                                         |
| ----------------- | ------------------------------------------------------------ |
| 가이드 오버레이   | 촬영 시 화면에 표시되는 두피 위치 안내 실루엣                |
| 비교 슬라이더     | 두 시점의 사진을 좌우로 슬라이드하며 비교하는 UI             |
| 면책 고지         | "AI 참고용 분석 · 의료 진단 아님" 등 의료 면책 문구         |
| 병원 권유 팝업    | Grade 4~5 결과 시 전문의 상담을 안내하는 모달                |
